Tab

Switch displayed content. Six variants - underline, pill, segmented, solid, filled, and icon - in three sizes.

<x-tab-group variant="filled" size="small" stretch :tabs="[
    ['key' => 'monthly', 'label' => 'Monthly', 'active' => true],
    ['key' => 'yearly', 'label' => 'Yearly'],
]" />

<x-tab-group variant="segmented" size="large" :tabs="[
    ['label' => 'Card', 'icon' => 'SquaresFour', 'active' => true],
    ['label' => 'list', 'icon' => 'ListBullets'],
]" />

Click or use arrow keys to switch tabs.

Button

The definition of a button component refers more to its interactive behavior than to the shape and elements of the component.

That is, we define the UI element used for action as a button.

Buttons can be icons, text, and combinations thereof.

Features

  • Size: Small, Medium, Large.
  • Variant: Borderless, Gray, Outline, Filled.
  • State: Default, Hover, Disabled.
  • Left Icon: True, False
  • Right Icon: True, False
  • Text: True, False

In addition to the button styles provided by default, you can change the button color, rounded corners, etc. to create more button styles, but you need to comply with the design system.

Correlation

Instances within the Button are replaceable with any instance except itself.

Text

The purpose of making text components is to make it more extensible.

Features

  • Count: The number of text in the instance.
  • State: Default, hover, No interaction.
  • Vertical: Arrange vertically or horizontally.

Correlation

  • Use text styles to change the text size.
  • Change the text color using color variables and color styles.
  • Text components are often paired with and Content components.

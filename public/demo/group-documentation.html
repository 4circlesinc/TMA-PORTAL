Group

The function of the Group component is to group any instances.

The text group is provided by default. You can replace the text with other instance such as icon, avatar, button, IconText, etc., or combine them at will.

Features

  • Count: Number of instances in the Group.
  • Vertical: Arrange vertically or horizontally.
  • Reverse: Sequential or reverse.

Correlation

Instances inside the Group can be replaced with any instance except itself.

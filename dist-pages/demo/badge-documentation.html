Badge

Badges are used to highlight information such as notifications and new messages.

Features

Type: Dot, Number.

Correlation

Used in the Icon component.

It can also be used alone.

When to use Number and when to use dot?

Number draw the user's attention more than Dot, and are more intrusive.

  • Use Number or Dot to determine the importance of information to users. Use Number for important information.
  • If the frequency of notifications is high, Number are not suitable.
  • Judgment based on the significance of the notification to the user, such as the increase in the number of followers, it is recommended to use Number, and the new function reminder is recommended to use Dot.
  • It is best to allow users to choose the notification method or turn off the notification.

The notification disappears after the user clicks it.

Please be cautious about the timing and method of notifications, which will have a huge impact on user experience.

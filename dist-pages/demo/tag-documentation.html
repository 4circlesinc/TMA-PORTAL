Tag

Tag components are typically used for options.

The difference between Tag and IconText is that Tag usually has a background.

Features

  • Type: Default, Left arrow, Right arrow.
  • State: Component’s interaction state. When previewing, this component supports changes in the interaction state.
  • Hide left icon.
  • Hide Right icon.

Correlation

  • Similar to IconText.
  • Similar to Status Badge.

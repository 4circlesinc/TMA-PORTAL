Input

The Input component is used to create various Inputs.

Input defaults to the Content instance, which can be replaced with other instances.

All forms demo

Interactive guidance

Form instances

Features

  • Type: 1 row, 2row vertical, 2 row horizontal, Textarea
  • State: Default, Hover, Focus, No interaction.

Correlation

Instances within the Input are replaceable with any instance except itself.

Examples

Because form elements such as Input and select fields often have the same style, it's easier to maintain them all using the Input component.

Any element that can Input text can be done with the input component.

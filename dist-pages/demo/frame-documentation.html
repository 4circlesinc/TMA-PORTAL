Frame

A combination of icon and text.

You can also replace icon with avatars, logos, pictures and other components.

The difference between Frame and IconText is that Frame can contains IconText instances, and Frame provides more layout structures.

The purpose of designing Frame components is to make them carriers of content, capable of supporting content with various layout structures.

Features

  • State: Component’s interaction state. When previewing, this component supports changes in the interaction state.
  • Vertical: Arrange vertically or horizontally.
  • Flip: Flip layout.

Correlation

  • Icon component
  • Text component
  • IconText component

Instances within the Frame can be replaced by any instance other than itself.
